☁️ Cloud Sync Settings

```
✅ Firebase connected! Your data syncs automatically across all devices in real time.
🔐 Sync Password

Set a password so only you can access your cloud data. All devices must use the same password.

🔄 Manual Sync

Push = send your local data to cloud (overwrites cloud). Pull = get cloud data to this device (overwrites local).

📱 How to use on multiple devices
  1. Open the app URL on any device
  2. Go to ☁️ Cloud Sync and enter the same password
  3. Click Pull from Cloud to get existing data
  4. From now on, all changes sync automatically!
```

📦 Export / Import Data

```
⬇️ Export — Download your data

Downloads all your tasks, members, and today items as a JSON backup file. Save it somewhere safe.

⬆️ Import — Restore from backup

Select a previously exported JSON file. This will merge with existing data — existing tasks are kept and new ones from the file are added (no duplicates).

💡 Multi-device tip: Export on Device A → send the JSON file to Device B → Import on Device B. Your data stays in sync manually this way.
```

📱 Install as Mobile App

📱 iPhone / iPad (Safari)
  1. Open this website in Safari
  2. Tap the Share button (box with arrow) at the bottom
  3. Scroll down and tap "Add to Home Screen"
  4. Tap Add — the app icon appears on your home screen
  5. Open it from home screen — it works like a native app!
🤖 Android (Chrome)
  1. Open this website in Chrome
  2. Tap the 3-dot menu (top right)
  3. Tap "Add to Home Screen" or "Install App"
  4. Tap Add — done!
💻 Desktop (Chrome / Edge)
  1. Look for the install icon (⊕) in the address bar
  2. Click it and select Install
  3. App opens in its own window — pin to taskbar
ℹ️ Your data is stored in the browser — it syncs across devices only if you use the same browser profile (e.g. Chrome signed in with same Google account).
